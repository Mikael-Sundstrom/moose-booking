Ett enkelt bokningssystem för WordPress där besökare kan boka dag, helg eller vecka.

== Description ==
Simple Booking är ett lättviktigt bokningsplugin för små webbplatser. Välj dag, helg eller veckobokning.

== Installation ==
1. Ladda upp `simple-booking`-mappen till `/wp-content/plugins/`.
2. Aktivera tillägget via menyn "Tillägg" i WordPress.
3. Använd blocket "Simple Booking" i din editor.
